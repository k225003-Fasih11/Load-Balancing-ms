# Load Balancing Management System (LBMS)

A full-stack Management Information System for monitoring, configuring, and analyzing load balancing infrastructure.

## Tech Stack
- **Frontend:** React 18, React Router, Chart.js, Axios, TailwindCSS
- **Backend:** Node.js, Express.js, JWT Auth, bcrypt
- **Database:** MongoDB (Mongoose)

## Features (15)
1. User Authentication (Register/Login/Logout)
2. Role-Based Access Control (Admin / Viewer)
3. Server Management — CRUD
4. Algorithm Configuration
5. Real-Time Load Dashboard
6. Request Routing Simulation
7. Traffic Analytics & Charts
8. Alert & Threshold Management
9. Incident Log
10. Performance Reports (exportable)
11. Algorithm Comparison Module
12. AI Traffic Predictor Panel
13. Server Health Monitor
14. User Activity Log
15. Settings & Notification Preferences

## Setup

### Backend
```bash
cd backend
npm install
cp .env.example .env   # fill in your values
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```
